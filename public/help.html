Family Website Help

Addressbook

Entries are not linked to user accounts, but can be any place you wish.

The entry first name is linked to edit the entry itself.

The street address is linked to a Google Map of the location.

The phone numbers are linked with the prefix "tel:" and email with "mailto:"

Album

Each active site user will have a block for their uploads.

The first uploaded image will be used for that user's main album entry.

The album can contain any file besides ".txt"

Images (gif/jpg/png) are shown as a linked icon sized version.

If the upload is not an image, currently these formats are recognized with their own icon: .mp3, .mp4 and .pdf

To add a caption for an upload, just type it into the slot below the file and hit the enter key, and it will be saved.

Calendar

This rotating calendar is for one-year anniversaries.

The "sticky" flag adds this event as a reminder to the site chat, year-round.

The event notes only appear on the chat during the month or if the sticky flag is applied.

Chat

For this demo, the time stamps are all in the Pacific time-zone, but can be anywhere for a particular site.

Angle brackets are not allowed. Please use the & lt ; and & gt ; HTML entities instead (with no spaces).

Cookbook

Entries are listed according to their "recipe type", like "Appetizer" for instance.

To add a new recipe, click the "NEW" link in the categories list.

The Ingredients and Instructions input boxes can be resized in most browsers.

When viewing an entry, clicking the title will edit the entry itself.

Logout

Please use this when logged-in at a public place.

On your home device, you can stay logged in as long as the browser cookie is present.

In the event that the site has to be restarted or its session files cleared, you will have to re-authenticate.

Administrator Help

Visit the Log page to see a summary of activity and an additional set of admin buttons.

When a new user account is made, an album block and an addressbook entry are made, as well as an optional calendar entry.

There is no email component of signup/grant - the new account process is by personal communication between administrator and new user.

The history page date inputs will accept many common formats, including "yesterday", "today", and "11/26/18."
